About PhishRadar

PhishRadar is a machine learning-powered tool designed to help users identify potentially malicious or phishing websites instantly.

By analyzing URL structures and patterns commonly used in phishing attacks, the system predicts whether a link is safe or suspicious.

This project was built as a Final Year Project to explore the intersection of cybersecurity and machine learning, with the goal of making the internet a safer place.

How it works

• URL is cleaned and processed
• Features are extracted from the link
• A trained ML model classifies it as safe or phishing
• Confidence score is returned to the user
